1. 특정 경로를 기준으로 내용이 포함된 파일 이름 찾기
grep -nrw <경로> -e <찾을 내용>

    -n : 라인 번호 출력
    -r : 경로를 기준으로 재귀적으로 탐색
    -w : 완전히 일치하는 걸 탐색
    -e : 정규표현식으로 찾기
 
2. 특정 경로를 기준으로 파일 찾기
    1. find <경로> -name <파일 명>


3. stdin 버퍼의 내용을 클립보드로 복사
    echo "Hello World" | xsel -i -b

    -i : stdin 버퍼의 내용을 읽음
    -b : 시스템 클립보드로 복사
